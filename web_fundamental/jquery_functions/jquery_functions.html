Click

Click the button to trigger an alert event.

Hide

Click to hide the paragraph element.

Toggle

Click to display or hide the paragraph.

Slideup and SlideDown

Click to display or hide the paragraph in sliding motion.

SlideToggle

Click to display or hide the paragraph in sliding motion.

FadeIn and FadeOut

Click to display the paragraph by fading them to opaque.

AddClass

Click to add a class to the h1 tag to change the color.

Before

Click to insert the word "The" before each the h1 elements.

After

Click to insert content, specified by the parameter, after each element in the set of matched elements.

Append

Click to insert "Wow!!"to the end of paragraph element.

HTML

Click to get the HTML contents paragraph element changed to new content.

Attr

Click to get the value of an attribute for the first element in the set of matched elements or set one or more attributes for every matched element.

Val

Click to get the current value of the first element in the set of matched elements or set the value of every matched element.

Text

Click to get the combined text contents of each element in the set of matched elements, including their descendants, or set the text contents of the matched elements.
