Click

Click the button to trigger an event.

Hide

Click to hide the element.

Show

Click to display matched elements.

Toggle

Click to display or hide the matched elements.

Slidedown

Click to display the matched elements with a sliding motion

Slideup

Click to hide the matched elements with a sliding motion.

Slidetoggle

Click to display or hide the matched elements with a sliding motion.

Fadein

Click to display the matched elements by fading them to opaque.

Fadeout

Click to hide the matched elements by fading them to transparent.

AddClass

Click to add the specified class(es) to each element in the set of matched elements.

Before

Click to insert content, specified by the parameter, before each element in the set of matched elements.

After

Click to insert content, specified by the parameter, after each element in the set of matched elements.

Append

Click to insert content, specified by the parameter, to the end of each element in the set of matched elements.

HTML

Click to get the HTML contents of the first element in the set of matched elements or set the HTML contents of every matched element.

Attr

Click to get the value of an attribute for the first element in the set of matched elements or set one or more attributes for every matched element.

Val

Click to get the current value of the first element in the set of matched elements or set the value of every matched element.

Text

Click to get the combined text contents of each element in the set of matched elements, including their descendants, or set the text contents of the matched elements.
